Update, Insert Or Delete Data

  • Note: your query must contain respective schema name which you mentioned while connection your data sources.

  • Note: your query must not contain any reserved word.



Query Examples

INSERT:

INSERT INTO (schema_name).(table_name) VALUES('value1', 'value2', 'value3');

UPDATE:

UPDATE (schema_name).(table_name) SET column_name1='column_value1', column_name2='column_value2' WHERE column_value='value';

DELETE:

DELETE FROM (schema_name).(table_name) WHERE column_value='value';
